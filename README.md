# SmartFit - AI Fitness Band Landing Page

A modern, responsive product landing page for "SmartFit," a conceptual AI-powered fitness band. This project demonstrates semantic HTML, responsive CSS (Flexbox/Grid), and interactive JavaScript functionality, including a dark mode toggle and form validation.

##  Features

* **Responsive Design:** Fully adaptable layout for Desktop, Tablet, and Mobile devices.
* **Dark Mode Support:** Toggle between Light and Dark themes (preferences saved in LocalStorage).
* **Smooth Scrolling:** Polished navigation experience with header offset calculations.
* **Scroll Animations:** Elements fade in and slide up as you scroll down the page.
* **Interactive UI:**
    * Sticky Navigation Bar with glassmorphism effect.
    * Hover effects on cards and buttons.
    * Working contact form validation (client-side).

##  Technologies Used

* **HTML5:** Semantic structure.
* **CSS3:** Flexbox, Grid, CSS Variables, Media Queries, Transitions.
* **JavaScript (Vanilla):** DOM manipulation, Event Listeners, LocalStorage API, Intersection Observer API.
* **Google Fonts:** Poppins typeface.
